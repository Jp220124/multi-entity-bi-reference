Phase 1 · The Four Foundation Agents (per Service Agreement)

Agent 1 · Classification & Routing

L1 / Haiku. Ingests raw inputs, classifies each by entity, category, priority. Routes to downstream agents. High-volume, low-latency.

Agent 2 · Cross-Entity Analyzer

L2 / Sonnet. Finds patterns only visible when hospitality, retail, real estate data are compared. Schema validation on every output.

Agent 3 · Strategic Synthesis

L3 / Opus + Extended Thinking. Portfolio-level decisions. Pattern recognition, reversal detection, contradictions across entities.

Agent 4 · Alerting / Output Delivery

Shared utility for Agents 1–3. Formats & delivers to Slack / email. Threading, retry logic, dead-letter logging.

Phase 1 scope note: These four agents establish the foundation architecture — the classification, analysis, synthesis, and delivery infrastructure on which business-domain agents (revenue, margin, inventory, executive brief, etc.) are built in Phase 2 onward. The input sources, classification taxonomy, and exact output channels are finalised jointly during Week 1 kickoff.
